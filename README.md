 ## **Реализация метода стилизованного мультяшного рендеринга (Cartoon / Cel Shading)**
 
Выполнила Мерненко Виктория Сергеевна, группа ИВТ-365

## Возможности

- **Загрузка моделей:** Поддержка формата `.obj` через библиотеку Assimp.
- **Интерактивность:**
  - Открытие файла.
  - Регулировка толщины обводки.
  - Автоматическое вращение модели.

## Управление

| Клавиша / Действие        | Функция                              |
| ------------------------- | ------------------------------------ |
| **Колесо мыши**           | Приближение / Отдаление камеры       |
| **Зажатое Колесо + Мышь** | Вращение камеры вокруг объекта       |
| **Alt + Колесо мыши**     | Изменение толщины обводки            |
| **O**                     | Открыть новую модель                 |
| **R**                     | Включить / Выключить вращение модели |

##  Как собрать и запустить

Для сборки требуется **Microsoft Visual Studio** c установленным компонентом "Desktop development with C++".
### Способ 1: Через командную строку (CLI)

1. Открыть **x64 Native Tools Command Prompt for VS 2022**.
2. Перейдите в папку проекта:
   ```cmd
   cd путь\к\папке\проекта
   ```
3. Выполнить следующую команду сборки:

```cmd
cl /MDd /EHsc /std:c++17 /I dependencies\include /I src CelShaiding\main.cpp /D GLEW_STATIC /link /LIBPATH:dependencies\lib glew32s.lib glfw3.lib assimp-vc143-mtd.lib opengl32.lib user32.lib gdi32.lib shell32.lib comdlg32.lib /NODEFAULTLIB:LIBCMT /NODEFAULTLIB:MSVCRT /OUT:MyApp.exe && copy /Y dependencies\lib\assimp-vc143-mtd.dll assimp-vc143-mtd.dll
```

3. Запустить файл `build.bat`

## Используемые библиотеки

*   **GLFW** - Создание окна и обработка ввода.
*   **GLEW** - Загрузка расширений OpenGL.
*   **GLM** - Математика (векторы, матрицы).
*   **Assimp** - Загрузка 3D-моделей.
